# branvier_template

 Branvier project template

```txt
lib/
├─ env.dart
├─ main.dart
└─ app/
   ├─ app_module.dart
   ├─ app_routes.dart
   ├─ app_widget.dart
   ├─ module/
   │  └─ {module_name}/
   │     ├─ data/
   │     │  ├─ models/
   │     │  ├─ repositories/
   │     │  └─ services/
   │     ├─ view/
   │     │  └─ {view_name}/
   │     │     ├─ {view_name}_controller.dart
   │     │     └─ {view_name}_page.dart
   │     └─ {module_name}_module.dart
   └─ shared/
      ├─ models/
      ├─ repositories/
      ├─ services/
      ├─ sources/
      └─ utils/
```
